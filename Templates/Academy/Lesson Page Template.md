# Lesson Page Template

Use this template when creating a new Academy lesson page.

## Required metadata

- Course name
- Domain number
- Domain title
- Lesson number
- Lesson title
- Page title
- Related library topics
- Lesson part list

## Page title format

Use the current lesson first.

Example:

Domain 3 Lesson 4 Inclusive Design and Usability - CPACC Preparation Course

## H1

Use one H1.

Example:

Domain 3 Lesson 4: Inclusive Design and Usability

## Required page sections

### Lesson Overview

Briefly explain what the learner will understand or be able to do after the lesson.

### Related Library Topics

List only topics that directly support this lesson.

### Part Controls

Provide Previous Part, Related, and Next Part when applicable.

### Lesson Content

Use H3 headings for lesson parts.

### Progress

Provide course, domain, lesson, and part progress when available.

### Navigation

Provide navigation to the previous lesson, next lesson, domain, and home when applicable.

## Accessibility checklist

- One H1.
- Logical heading order.
- Page title identifies the lesson.
- Controls have clear names.
- Links describe destinations.
- No color-only state.
- Reflows at 320 CSS pixels.
- Works with keyboard only.
- Works with JAWS, NVDA, and VoiceOver.
