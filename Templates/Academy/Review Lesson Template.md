# Review Lesson Template

Use this template when creating a domain review lesson.

## H1

Use the current domain and review lesson title.

Example:

Domain 5 Review: Accessibility Laws, Standards, and Management

## Recommended sections

### Review Overview

Explain the purpose of the review.

### Domain Mental Model

Summarize how the domain fits together.

### Key Concepts

Connect the major ideas across lessons.

### Scenario Workshop

Use applied scenarios to test reasoning.

### Professional Reasoning Patterns

Explain how accessibility professionals analyze situations.

### Common Misconceptions and Exam Traps

Clarify likely points of confusion.

### Connections Across Domains

Show how this domain connects to earlier learning.

### Exam Focus

Identify what learners should recognize on an assessment.

### Summary

Close with a plain-language synthesis.

## Accessibility checklist

- One H1.
- Sections are navigable by heading.
- Scenario text is clear and concise.
- No hidden answer content without accessible controls.
- Progress is available in text.
- Works at 320 CSS pixels without horizontal scrolling.
