# Lesson Navigation Component

## Purpose

The lesson navigation component provides consistent movement through lesson parts and course pages.

## Common controls

- Previous Part
- Next Part
- Related
- Previous Lesson
- Next Lesson
- Home

## Accessibility requirements

- Controls use clear names.
- Disabled states are clear visually and programmatically.
- Focus moves predictably after activation.
- The component works with keyboard only.
- The component reflows into a single column on narrow screens.

## Focus rule

When a part changes, focus should move to the heading or start of the newly displayed content.

## Why this component exists

Lesson navigation is repeated across Academy pages. A shared component keeps movement predictable and reduces learner effort.
