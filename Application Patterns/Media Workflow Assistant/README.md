# Media Workflow Assistant Pattern

## Purpose
The Media Workflow Assistant is the Open Door Design pattern for applications that transform media into accessible outcomes.

Users provide media.
The application identifies the media.
The application recommends meaningful workflows.
Users choose outcomes, not tools.

## Core Principles
- Goal-oriented interaction.
- Automatic media inspection.
- Outcome before technology.
- Progressive disclosure.
- Keyboard-first.
- Screen reader first.
- Automation with user control.

## Application Lifecycle
1. Launch to a drop zone.
2. Inspect media.
3. Identify capabilities.
4. Recommend workflows.
5. Execute selected workflow.
6. Present results and suggested next actions.
