# Library Topic Pattern

## Purpose

Library topics provide optional reference support for learners who want more context.

## Placement

Related Library Topics should appear near the start of a lesson, after the lesson is identified and before the main lesson flow becomes deep.

## Naming

Topic names should be specific and understandable.

Avoid generic labels such as More, Related, or Extra.

## Relevance

Each related topic should have a clear relationship to the current lesson.

Do not include a topic merely because it exists in the library.

## Screen reader behavior

The related topics section should have a clear heading.

Links should describe their destinations without requiring surrounding text.

## Optional nature

The section should not make the learner feel that the library is required to complete the lesson.

## Why this pattern exists

Optional reference material is valuable, but it can overwhelm the learner if inserted directly into the lesson. The library pattern keeps reference material available without disrupting the instructional path.
