# Accessible Folder Documentation Example

## Good folder description

Root files:

- README.md
- .gitattributes

Root folders:

- Archive
- Assets
- Components
- Examples
- Patterns
- Philosophy
- Resources
- Standards
- Templates

## Why this is preferred

This format is faster and clearer for screen reader users than ASCII tree diagrams. It avoids repeated punctuation and line-drawing symbols that are announced as symbol noise.
