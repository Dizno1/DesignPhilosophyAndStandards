# Accessible Multimedia Carousel Reference Implementation

This example preserves the working carousel demo as a reusable Open Door Design reference implementation.

Use this folder when a new website or application needs the same carousel behavior. The page links to the shared component files in `Components/Carousel` so future improvements to the component remain centralized.

## What this example demonstrates

- Three independent carousel regions on one page.
- A site-wide persistent Pause motion preference.
- Focus moving to the new slide summary after Previous or Next.
- Concise live-region announcements.
- Video playback pausing carousel motion.
- Form validation inside a carousel slide.
- Reflow-friendly layout.
- iPhone and VoiceOver lesson: pausing motion turns the carousel into an exploration mode and dramatically improves usability.
